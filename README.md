# 3D Web Structure Analyzer

ブラウザのみで動作する3D弾性骨組構造解析Webアプリケーション。

## セットアップ

```bash
npm install
```

## 起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開く。

## ビルド

```bash
npm run build
```

`dist/` に静的ファイルが生成される。任意のHTTPサーバーで配信可能。

## テスト

```bash
npm test
```

## 対応機能

### モデル構築
- 節点の作成・移動・削除（キャンバスクリック / ドラッグ）
- 部材の作成・削除（2点指定）
- 支持条件の設定（自由 / 固定の切替）
- 材料特性の設定（名称・ヤング係数 E・せん断弾性係数 G・ポアソン比 v・熱膨張係数）
- 断面特性の設定（名称・断面積 A・ねじり定数 Ix・断面二次モーメント Iy, Iz・せん断面積比 ky, kz）
- 材料・断面ライブラリからのプリセット追加
- 部材端バネ（剛・ピン・回転バネ）の設定と静的縮約
- 両端ピン部材向けの一端捻り拘束（部材軸がグローバルX/Y/Z軸に平行な場合。実体は該当端節点の軸方向回転DOF固定のため、同じ節点を共有する他部材にも影響）
- 同一変位カップリング（master-slave DOF マッピング）
- 2Dフレーム解析モード（X-Z平面、uy/rx/rz を自動拘束、全節点 Y=0 を検証）
- X-Z 2Dモードへ切り替える前に節点Y座標を0へ明示変換する支援操作
- 部材コード角の設定
- 節点荷重（Fx, Fy, Fz, Mx, My, Mz）
- 部材荷重（集中荷重・等分布荷重・CMQ荷重）
- 荷重ケースと荷重組合せ（ケース別荷重、係数付き組合せ解析）

### インポート / エクスポート
- FrameModelMaker-Web 形式（FrameJsonDocument）のインポート
- FrameModelMaker-Web 形式は3Dモデルとして読み込み（2DモードはProjectFile JSONで保存/復元）
- JSON形式のエクスポート / インポート
- サンプルモデル読込
- IndexedDBオートセーブ

### 解析
- 線形弾性・微小変形の3D弾性骨組解析
- 6自由度/節点（ux, uy, uz, rx, ry, rz）
- 12x12 Timoshenko梁要素剛性マトリクス（軸力・ねじり・2軸曲げ）
- せん断変形を考慮（Timoshenko梁理論）
- 3D座標変換（部材方向・コード角対応）
- 部材端バネの静的縮約（ピン・回転バネ）
- 同一変位カップリング（master-slave DOF、slave側拘束の master 伝播を含む）
- Web Workerによる非同期解析（UI非ブロック）
- LDLt分解による連立方程式求解
- 特異剛性マトリクス発生時の不安定モデル診断（候補節点DOF、剛性ゼロDOF、両端ピン部材の表示）

### 結果表示
- Three.js による3D変形図（倍率調整可能）
- 断面力図（N, Vy, Vz, Mx, My, Mz）
- CMQ moy/moz の放物線バブル補正による断面力図反映
- 数値結果テーブル（節点変位・反力・部材端力）
- 解析レポート出力（Markdown / CSV / 印刷用PDF）
- カップリング連成時の反力表示（代表行表示・重複防止）
- 荷重オーバーレイ（節点力・モーメント・集中荷重・等分布荷重・CMQ端部力/モーメント）
- 支持条件の可視化
- 荷重 / 支持の表示切替
- 節点番号・部材番号の表示切替
- ライト / ダークテーマ対応

## 操作方法

| 操作 | 動作 |
|------|------|
| ツールバーで「節点」選択 → キャンバスクリック | 節点作成（Z=0 平面） |
| ツールバーで「部材」選択 → 2つの節点をクリック | 部材作成 |
| ツールバーで「支持」選択 → 節点クリック | 支持条件切替（自由 ↔ 全固定） |
| ツールバーで「節点荷重」選択 → 節点クリック | 節点荷重追加 |
| ツールバーで「部材荷重」選択 → 部材クリック | 部材荷重追加 |
| ツールバーで「選択」 → 節点/部材クリック | 選択（右パネルでプロパティ・荷重編集） |
| Shift + クリック | 複数選択（節点・部材の追加選択） |
| 選択ツール + ドラッグ | 節点移動（元のZ座標を保持） |
| Delete / Backspace | 選択要素の削除（フォーム入力中は無効） |
| Escape | 選択解除・操作キャンセル |
| マウスホイール | ズーム |
| 右ドラッグ | 回転（OrbitControls） |
| 中ボタンドラッグ | パン |
| ▶ 解析実行 | 解析を実行し結果を表示 |

## 符号規約

- **座標系**: 右手系（X, Y, Z）、Z上正
- **部材ローカル軸**: x = i端→j端方向、y/z = 参照ベクトルとコード角で決定
- **軸力**: 正 = 引張
- **荷重方向**: localX = i端→j端方向、localY/localZ = 部材ローカル軸に沿う
- **部材端力**: ローカル座標系で表示（Ni, Vyi, Vzi, Mxi, Myi, Mzi, Nj, ...）
- **断面力図**: 引張側に描画

## 技術スタック

- TypeScript + React + Vite
- Three.js + OrbitControls（3D描画）
- Zustand（状態管理）
- Web Worker（解析処理）
- IndexedDB（永続化）
- Vitest（テスト）

## 制約事項

- 線形弾性・微小変形のみ対応
- 部材荷重は集中荷重・等分布荷重・CMQ のみ対応（台形分布荷重は未対応）

- 台形分布荷重 / 部分分布荷重
- Undo / Redo
