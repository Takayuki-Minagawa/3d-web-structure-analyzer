# 2D Frame Structure Analyzer

ブラウザのみで動作する2Dフレーム構造解析Webアプリケーション。

## セットアップ

```bash
npm install
```

## 起動

```bash
npm run dev
```

ブラウザで `http://localhost:5173` を開く。

## ビルド

```bash
npm run build
```

`dist/` に静的ファイルが生成される。任意のHTTPサーバーで配信可能。

## テスト

```bash
npm test
```

## 対応機能

### モデル構築
- 節点の作成・移動・削除（キャンバスクリック / ドラッグ）
- 部材の作成・削除（2点指定）
- 支持条件の設定（自由 / ピン / 固定 / ローラー）
- 材料・断面特性の設定
- 節点荷重（Fx, Fy, Mz）
- 部材荷重（集中荷重、等分布荷重）

### 解析
- 線形弾性・微小変形の2Dフレーム解析
- Euler-Bernoulli梁理論ベース
- Web Workerによる非同期解析（UI非ブロック）
- LDLᵀ分解による連立方程式求解

### 結果表示
- 変形図（倍率調整可能）
- 軸力図
- せん断力図
- 曲げモーメント図
- 数値結果テーブル（節点変位・反力・部材端力）

### 永続化
- IndexedDBオートセーブ
- JSON形式のエクスポート / インポート
- サンプルモデル読込

## 操作方法

| 操作 | 動作 |
|------|------|
| 左ツールバーで「節点」選択 → キャンバスクリック | 節点作成 |
| 左ツールバーで「部材」選択 → 2つの節点をクリック | 部材作成 |
| 左ツールバーで「支持」選択 → 節点クリック | 支持条件切替（自由→ピン→固定→ローラー→自由） |
| 左ツールバーで「節点荷重」選択 → 節点クリック | 節点荷重追加 |
| 左ツールバーで「部材荷重」選択 → 部材クリック | 部材荷重追加 |
| 左ツールバーで「選択」 → 節点/部材クリック | 選択（右パネルでプロパティ・荷重編集） |
| 選択 → ドラッグ | 節点移動 |
| Delete / Backspace | 選択要素の削除 |
| マウスホイール | ズーム |
| 中ボタンドラッグ | パン |
| ▶ 解析実行 | 解析を実行し結果を表示 |

## 符号規約

- **座標系**: X右正、Y上正、Rz反時計回り正
- **軸力**: 正 = 引張
- **荷重**: 正方向 = 座標軸正方向（localXはi端→j端方向、localYはその90°反時計回り）
- **部材端力**: ローカル座標系で表示

## 技術スタック

- TypeScript + React + Vite
- Zustand（状態管理）
- Canvas 2D（描画）
- Web Worker（解析処理）
- IndexedDB（永続化）
- Vitest（テスト）

## 制約事項

- 線形弾性・微小変形のみ対応
- せん断変形は考慮しない
- 2D平面内解析のみ

## 既知の未対応項目

- 座屈解析 / 動的解析

- 部材端剛域 / 部材端リリース
- 台形分布荷重 / 部分分布荷重
- Undo / Redo
